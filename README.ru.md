# Text Quest Reader

![Unity](https://img.shields.io/badge/engine-Unity-000000?logo=unity&logoColor=white)
![Language](https://img.shields.io/badge/language-C%23-blue)
![License](https://img.shields.io/badge/license-MIT-green)

Ридер текстовых квестов для запуска ветвящихся сюжетных сценариев, вдохновлённый механиками игры «Космические Рейнджеры».

Вместе с инструментом [Quest Editor](https://github.com/albruevich/QuestEditor_Builds) образует систему для создания и запуска собственных текстовых квестов.

В отличие от оригинального редактора, привязанного к конкретной игре, эта система позволяет создавать и использовать собственные квесты в своих проектах.

Поддерживает логику локаций, переходы, параметры, а также изображения и звуки.

## О проекте

Проект написан на C# с использованием Unity.

Является open source и может быть использован:
- для прохождения квеста "Asteroid Station"
- чтобы понять, как использовать квесты, созданные в Quest Editor
- как основа для создания собственного ридера текстовых квестов с кастомным UI

## Структура квеста

Каждый квест хранится как отдельная папка внутри:

Assets/Resources/Quests/

Внутри этой папки каждый квест имеет свою директорию:

YourQuest/

Структура выглядит следующим образом:

<img src="docs/quest_structure.png" width="240">

### Содержимое

- `quest.json` — основной файл квеста (содержит всю логику и данные, обязателен)
- `Images/` — изображения (необязательно)
- `Sounds/` — звуки (необязательно)
- `Musics/` — музыка (необязательно)

Если дополнительные папки отсутствуют, квест всё равно будет работать с текстом и логикой.

⚠️ Эта структура автоматически создаётся в Quest Editor.  
Её не нужно создавать вручную.

Чтобы использовать квест, просто поместите его папку в:

Assets/Resources/Quests/

Ридер автоматически его обнаружит.

💡 Пример:  
Assets/Resources/Quests/AsteroidStation/

---

## Важно

Имя квеста внутри `quest.json`:

```json
"questName": "YourQuest"
```

должно совпадать с названием папки квеста.

## Как запустить

1. Откройте папку проекта в Unity Hub  
   (папку, содержащую `Assets`, `Packages` и `ProjectSettings`)

2. Откройте сцену:

Assets/_Scenes/MainScene.unity

3. Нажмите Play

---

## Быстрая проверка

После запуска:
- выберите квест (например, "Asteroid Station")
- нажмите "Start Selected Quest"

Если всё работает — квест запустится.

## Создание квестов

Квесты создаются с помощью отдельного инструмента — [Quest Editor](https://github.com/albruevich/QuestEditor_Builds).

👉 В репозитории редактора есть полная документация по созданию и экспорту квестов.

Редактор позволяет визуально создавать параметры, локации, переходы и структуру квеста, а затем экспортировать его в виде готовой папки, которую можно сразу использовать в этом ридере.

---

## Локализация

Ридер поддерживает несколько языков.

Можно добавить файлы:
- `quest_en.json`
- `quest_uk.json`

Если локализованный файл отсутствует — используется `quest.json`.

См. и редактируйте:
Localization.cs, LocKeys.cs, SettingsPanel.prefab, SettingsPanel.cs
