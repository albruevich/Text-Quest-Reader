# Text Quest Reader

![Unity](https://img.shields.io/badge/engine-Unity-000000?logo=unity&logoColor=white)
![Language](https://img.shields.io/badge/language-C%23-blue)
![License](https://img.shields.io/badge/license-MIT-green)

Рідер текстових квестів для запуску розгалужених сюжетних сценаріїв, натхненний механіками гри «Космічні Рейнджери».

Разом з інструментом [Quest Editor](https://github.com/albruevich/QuestEditor_Builds) утворює систему для створення та запуску власних текстових квестів.

На відміну від оригінального редактора та подібних систем, прив’язаних до конкретної гри або платформи, ця система дозволяє використовувати власні квести в будь-яких проєктах.

Підтримує логіку локацій, переходи, параметри, а також зображення та звуки.

## Про проєкт

Проєкт написаний на C# з використанням Unity.

Є open source і може використовуватися:
- для проходження квесту "Asteroid Station"
- щоб зрозуміти, як використовувати квести, створені в Quest Editor
- як основа для створення власного рідера текстових квестів з кастомним UI

## Структура квесту

Кожен квест зберігається як окрема папка в:

Assets/Resources/Quests/

У цій папці кожен квест має свою директорію:

YourQuest/

### Вміст

- `quest.json` — основний файл квесту (містить усю логіку та дані, обов’язковий)
- `Images/` — зображення (необов’язково)
- `Sounds/` — звуки (необов’язково)
- `Musics/` — музика (необов’язково)

Якщо необов’язкові папки порожні, квест усе одно працюватиме лише з текстом і логікою, але без звуку та зображень.

⚠️ Ця структура автоматично створюється в Quest Editor. Її не потрібно створювати вручну.

Щоб використати квест:
1. Помістіть його папку в `Assets/Resources/Quests/`
2. Додайте назву папки квесту до `_Settings/Quest Folders List`

Після цього рідер знайде та завантажить квест.

## Важливо

Назва квесту в `quest.json`:

```json
"questName": "YourQuest"
```

має збігатися з назвою папки квесту.

---

## Як запустити

1. Відкрийте папку проєкту в Unity Hub  
   (папку, що містить `Assets`, `Packages` і `ProjectSettings`)

2. Відкрийте сцену:

Assets/_Scenes/MainScene.unity

3. Натисніть Play

---

## Швидка перевірка

Після запуску:
- оберіть квест (наприклад, "Asteroid Station")
- натисніть "Почати Обраний Квест"

Якщо все працює — квест запуститься.

## Створення квестів

Квести створюються за допомогою окремого інструменту — [Quest Editor](https://github.com/albruevich/QuestEditor_Builds).

👉 У репозиторії редактора є повна документація щодо створення та експорту квестів.

Редактор дозволяє візуально створювати параметри, локації, переходи та структуру квесту, а потім експортувати його як готову папку, яку можна одразу використовувати в цьому рідері.

## Локалізація

Рідер підтримує кілька мов.

Можна додати файли:
- `quest_en.json`
- `quest_uk.json`

Якщо локалізований файл відсутній — використовується `quest.json`.

Див. і редагуйте:
Localization.cs, LocKeys.cs, SettingsPanel.prefab, SettingsPanel.cs
