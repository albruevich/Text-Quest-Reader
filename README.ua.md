# Text Quest Reader

![Unity](https://img.shields.io/badge/engine-Unity-000000?logo=unity&logoColor=white)
![Language](https://img.shields.io/badge/language-C%23-blue)
![License](https://img.shields.io/badge/license-MIT-green)

Рідер текстових квестів, натхненний механіками гри «Космічні Рейнджери».

Разом з інструментом Quest Editor утворює систему для створення та запуску власних текстових квестів.

Підтримує логіку локацій, переходів і параметрів, а також роботу з зображеннями та звуками.

## Про проєкт

Проєкт написаний на C# з використанням Unity.

Є open source і може використовуватися:
- для проходження готового квесту "Asteroid Station"
- для вивчення структури та архітектури проєкту
- як основа для створення власного рідера текстових квестів з кастомним UI

---

## Структура квесту

Кожен квест зберігається як окрема папка з такою структурою:

Assets/Resources/Quests/YourQuest/
    Images/
    Sounds/
    Musics/
    quest.json

- `quest.json` — основний файл квесту (JSON)
- `Images/`, `Sounds/`, `Musics/` — папки ресурсів (необов’язково)

Папки можуть бути порожніми — у цьому випадку квест міститиме лише текст і логіку.

---

## Важливо

Назва квесту в `quest.json`:

```json
"questName": "YourQuest"
```

має збігатися з назвою папки квесту.

## Створення квестів

Квести створюються за допомогою окремого інструменту — Quest Editor.

👉 Репозиторій редактора: [посилання]

Редактор дозволяє візуально створювати параметри, локації, переходи та структуру квесту, а потім експортувати його у формат, сумісний із цим рідером.

---

## Локалізація

Рідер підтримує кілька мов.

Можна додати файли типу:
- `quest_en.json`
- `quest_uk.json`

Якщо локалізований файл відсутній — використовується `quest.json`.

Див. також `Localization.cs` та `LocKeys.cs`.
