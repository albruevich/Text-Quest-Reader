Если вам приходилось играть в Космические Рейнджеры, то вы не могли пройти мимо текстовых квестов. 

Этот ридер текстовых квестов воссоздает похожую механику таких квестов. 

Проект написан на языке C# при помощи движка Unity. Является опер сорс и дает возможность посмотреть как построен проект. 

В основе проекта лежит папка Квеста, созданная при помощи моего Редактора Текстовых Квестов. 
Внутри этой папки находится JSON файл quest.txt, папка Images, Sounds, и Musics, которые предварительно создаются в Редакторе. 
Папку Квеста следует размещать по адресу Assets/Resources/Quests.
